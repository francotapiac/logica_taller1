Pasos para correr el programa:
1-Abrir carpeta "src"
2-Ejecutar el archivo "viewMain.py"
3-Aparecerá un menú con dos botones que indica dos opciones:
	•Consulta regular: Se pregunta por los factores de riesgo, luego se preguntan por los síntomas ordenadas según factores de riesgo, dando como resultado la enfermedad con un posible tratamiento.
	•Consulta por síntoma: Esta opción pregunta por todos los síntomas, dando como resultando una lista de posibles enfermedades. 

Contenido en "src":
Contiene 5 archivos y una carpeta
	•bd.pl -> Es la base de conocimiento de Prolog
	•codigo.py -> Contiene todas las funciones logicas, tanto de prolog como las propias de python
	•viewCS.py -> Vista de la opción consulta por sintomas
	•viewCR.py -> Vista de la consulta regular
	•viewMain.py -> Vitas principal que ejucuta el programa completo
	•imagenes -> Carpeta que contiene todas la imagenes usadas en las vistas
